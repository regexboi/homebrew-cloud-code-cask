# Cloud Code Homebrew Cask

Public Homebrew tap for installing prebuilt Cloud Code releases without publishing the main source repository.

## Install

```bash
brew tap regexboi/cloud-code-cask
brew install --cask cloud-code
```

Or install directly without tapping first:

```bash
brew install --cask regexboi/cloud-code-cask/cloud-code
```

## Notes

- Current release: `0.1.0`
- Current build: Apple Silicon (`arm64`) only
- The app is not notarized yet, so macOS may require an extra approval step on first launch
- Public repo: `https://github.com/regexboi/homebrew-cloud-code-cask`
